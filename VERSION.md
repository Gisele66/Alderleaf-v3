# Alderleaf Website — Version 3

**Created:** June 2026  
**Source:** Snapshot of [Alderleaf-v2](https://github.com/Gisele66/Alderleaf-v2) at initial v2 release.

## Purpose

Version 3 is an experimental branch of the Alderleaf site rebuild. Use it to try new layouts, stack changes, or design directions without affecting the stable v2 project.
